GETTING STARTED
---------------
Follow the next simple steps to import bitmap font to Unity.

0. Check the video tutorial https://www.youtube.com/playlist?list=PLCV-I26xBzv9Yk82-R4xA7Bn3IVEvYDbp

1. Create bitmap font with your favorite tool.
For example - this one online free http://kvazars.com/littera/
Export it to files fnt (xml format) and png (bitmap texture of your font)


2. Add font files to separate Unity assets folder.


3. Create custom font in Unity with "CJFinc: Bitmap font tools"
Run context menu command "CJFinc > RebuildBitmapFont" on font folder in Project view.


Now you can use new font with "UI text" object and any other that have font propperty.
